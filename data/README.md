# data/

Raw input data for the pipeline. These files are **not** shipped with the
repository because of their size and licensing; download them from the official
sources listed below and place them here before running the data-acquisition and
GTEx-integration steps.

## Expected layout

```
data/
├── gtex_raw/
│   ├── GTEx_Analysis_2017-06-05_v8_RNASeQCv1.1.9_gene_tpm.gct.gz   # GTEx v8 bulk gene TPM
│   ├── GTEx_Analysis_v8_Annotations_SampleAttributesDS.txt.txt     # GTEx sample attributes
│   ├── GTEx_v8_SubjectPhenotypes.txt                               # GTEx subject phenotypes
│   └── gtex_tpm.parquet                                            # generated by convert_gtex_parquet.py
└── full_cancer_data.csv                                            # optional single-CSV export of the database
```

## Sources

- **GTEx v8** files: GTEx Portal — https://gtexportal.org
- **TCGA** RNA-seq and clinical data: NCI Genomic Data Commons — https://portal.gdc.cancer.gov
  (downloaded and loaded into PostgreSQL by `python/gdc_download.py`)

## Note

The per-cohort, analysis-ready expression matrices (`dataset_TCGA_<cohort>.csv` and
`dataset_TCGA_<cohort>_vs_gtex.csv`) are produced by `python/create_ml_dataset.py`
and `python/gtex_integration.py`, and are written to
`python/ml_dataset_project_batches/`, not to this folder.
